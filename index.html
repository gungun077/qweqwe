웹페이지 만드는 법 배우기

HTML 기본 CSS 스타일링 JavaScript 소개

HTML 기본

HTML(하이퍼텍스트 마크업 언어)은 웹페이지의 구조를 정의합니다. 태그를 사용하여 제목, 단락, 링크 등을 정의할 수 있습니다.

<!DOCTYPE html>
<html>
<head>
    <title>페이지 제목</title>
</head>
<body>
    <h1>이것은 제목입니다</h1>
    <p>이것은 단락입니다.</p>
</body>
</html>

CSS 스타일링

CSS(캐스케이딩 스타일 시트)는 웹페이지를 꾸미는 데 사용됩니다. 색상, 글꼴, 레이아웃 등을 변경할 수 있습니다.

body {
    background-color: lightblue;
}
h1 {
    color: navy;
    font-family: Arial, sans-serif;
}

JavaScript 소개

JavaScript는 웹페이지에 상호작용을 추가합니다. 예를 들어, 버튼 클릭에 반응하거나 애니메이션을 만들 수 있습니다.

<script>
    function showAlert() {
        alert('안녕하세요, 세계!');
    }
</script>
<button onclick="showAlert()">클릭하세요</button>
